# 数据表格 datatable

主要用于展示大量结构化数据。

支持排序、固定列、固定表头、分页、自定义操作、单选多选等复杂功能。

### Restrict
- 'E'

## Arguments

| 属性 | 说明 | 类型 | 默认值 |
| --- | --- | --- | --- |
| closeOthers | 是否只允许同一时间展开一个group，值为`true`表示只允许展开一个，反之可同时展开多个 | `boolean` | `true` | 
| heading | group标题文本 | string |  |
| isOpen | 是否展开,值为`true`表示展开,反之收起 | `boolean` | `true` |
